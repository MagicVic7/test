# MagiChrono - Brain Storm V5.16

Norme de version : `Vnombre d'app.fichier`. Cette version est donc `V5.16`.

## Changements

- Titre et manifest renommés définitivement en **MagiChrono**.
- Footer : `By Magic Victor I www.magicvitor.be I V5.16`.
- Texte PIN discret : `Votre code annif/position contient au moins X chiffres`, où `X` est la position.
- Lecture API conservée avec proxy AllOrigins en secours.
- Secours manuel si l'API est vide : `PPxx` ou `PPJJMM` avec Appel d'urgence.
- Sécurité **START - STOP - RESET** réactivée pour Brain Storm : le forçage n'est armé qu'après cette séquence.
