# MagiChrono - Brain Storm V5.19

Correctif chrono qui se met en pause / se relance tout seul.

## Changement principal

- Ajout d'une protection anti-double-clic / anti-ghost-click sur le bouton Démarrer/Arrêter.
- Ajout d'un délai de sécurité de 650 ms après l'entrée dans le chrono : les anciens taps du PIN ne peuvent plus déclencher le chrono.
- Ajout d'un anti-double-clic sur Réinitialis./Tour.

Les corrections V5.18 sont conservées : Appel d'urgence relit l'API avant de switcher, encodage corrigé, proxy AllOrigins, branding MagiChrono, footer et START - STOP - RESET.
