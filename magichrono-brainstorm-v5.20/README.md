# MagiChrono - Brain Storm V5.20

Correctif annif dans les tours.

## Changement principal

- Les tours anniversaire relisent et sécurisent les valeurs depuis `birthday`, `full_peek` ou `text` juste avant le lancement et juste avant chaque tour.
- Le mode START - STOP - RESET reste actif.
- La protection anti-double-tap de V5.19 reste active.
