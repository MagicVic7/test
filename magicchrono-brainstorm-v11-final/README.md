# Magic Chrono - Brain Storm V11 Final

Corrige la lecture de `birthday` au format `(29/06)`.

Exemple supporté :

```json
{
  "text": "oliver tree",
  "birthday": "(29/06)",
  "full_peek": "39. oliver tree (29/06)",
  "Message1": "oliver tree",
  "Message2": "",
  "TargetPosition": "39",
  "timeUpdated": "2026-06-16 21:22:00"
}
```

- `TargetPosition` = 39 au stop.
- `birthday` ou `full_peek` = 29 puis 06 sur les deux premiers tours.
