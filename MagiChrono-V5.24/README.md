# MagiChrono V5.24

Changements :
- Position Time : mode 2 cartes remplacé par un mode multi-cartes.
- Appel d'urgence court sur l'écran PIN Position Time : ajoute une carte.
- Appel d'urgence maintenu : retour au mode 1 carte.
- Multi-cartes compatible avec Cut Deck : la carte du bas est entrée à la fin du PIN.
- Base stable V5.23 conservée : pavé numérique vérifié, Mnemonica corrigé, Brain Storm API, annif, proxy, encodage, anti-double tap, START-STOP-RESET.

Formats :
- Sans Cut Deck : 1 carte = 4 chiffres, 2 cartes = 6 chiffres, 3 cartes = 9 chiffres, etc.
- Avec Cut Deck : 1 carte + carte du bas = 6 chiffres, 2 cartes + carte du bas = 9 chiffres, etc.
