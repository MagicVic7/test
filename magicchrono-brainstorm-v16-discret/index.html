Tutoriel Number Force

Mode 1 nombre : code à 4 chiffres. Les 2 chiffres du milieu sont le nombre. Dernier chiffre : 9 = centièmes, 0 = somme.

Mode 2 nombres : code 0AABB0. AA est forcé avec le bouton Tour, BB au Stop.

Tutoriel Forcing Time

  1. Entrez un code à 6 chiffres au format MMSSCC.
  2. 000000 utilise le temps prédéfini dans les paramètres Forcing Time.
  3. Faites START - STOP - RESET pour armer le forçage.
  4. Les barres réseau passent de 4 à 3.
  5. Au prochain Stop, le chrono affiche le temps forcé.

Tutoriel Forcing Laps

Forcing Laps force une suite de centièmes sur les lignes de Tour.

  1. Par défaut le PIN est à 4 chiffres, par exemple 0102.
  2. Avec Appel d’urgence, le PIN passe au mode long : par défaut 6 chiffres, par exemple 010203.
  3. Fais START - STOP - RESET pour armer le forçage.
  4. À chaque pression sur Tour, l’app force le centième suivant.

Exemple : 010203, puis trois pressions sur Tour, donne par exemple 00 : 00, 01, 00 : 03, 02, 00 : 05, 03.

Tutoriel

Position Time est une application qui permet de forcer une position dans un jeu de cartes avec un chronomètre d'apparence tout à fait normale.

Fonctionnement :

  1. Configurer la carte secrètement (Écran PIN) : Utilisez le pavé numérique pour entrer le code de la carte choisie. Par défaut, tapez la valeur puis le code de la famille. Exemple : pour le 8 de Trèfle, tapez 083 puis un chiffre de camouflage.
  2. Confirmer discrètement : le pourcentage de batterie affiche le nombre que l'application va forcer.
  3. Armer le chronomètre : effectuez une séquence START - STOP - RESET.
  4. Réaliser le forçage : l'application ajuste les centièmes pour que la somme des secondes et des centièmes soit égale à la position.

Astuces de magicien :

  • Fausse mise en veille : touchez le haut de l'écran PIN.
  • Sécurité : si le spectateur attend trop longtemps, la cible est affichée dans les centièmes.
  • Sortie : touchez le coin supérieur gauche du chronomètre.

Nouvelles fonctionnalités

  • Stack personnalisé : import JSON ou CSV.
  • Mode 1 carte : code à 4 chiffres, forçage au Stop.
  • Mode 2 cartes : code à 6 chiffres ; bouton Appel d'urgence pour basculer discrètement. Seuls les points changent.
  • Forçage deux cartes : carte 1 au bouton Tour, carte 2 au Stop.
