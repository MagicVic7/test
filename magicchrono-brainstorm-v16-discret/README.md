# Magic Chrono - Brain Storm V16 Discret

Version basée sur V15 Proxy Hybrid, mais le texte PIN est moins explicite.

## API OK

- Position API `TargetPosition = 39`.
- Texte affiché : `Votre code position/annif contient au moins 39 chiffres.`
- Le nombre affiché dans la phrase est la position.
- L'anniversaire n'est plus affiché directement.

## API vide / erreur

- Secours position : `Votre code position contient au moins 4 chiffres.`
- Appel d'urgence : `Votre code annif contient au moins 6 chiffres.`

Le proxy AllOrigins est conservé comme secours de lecture API.
