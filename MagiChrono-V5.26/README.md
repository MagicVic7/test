# MagiChrono V5.26

Changements :
- Forcing Laps repris du fichier fourni : code de base `LLSS`, appui court urgence = +2 chiffres, appui long urgence = retour à 4 chiffres.
- Dernière paire `SS` = somme au Stop ; `SS=00` = vrai Stop chrono.
- `0000` utilise la suite prédéfinie des paramètres.
- Paramètre Forcing Laps simplifié : suite prédéfinie sous forme de couples, ex. `121400`.
- Base V5.25 conservée : multi-cartes Position Time + Cut Deck, Brain Storm API, Mnemonica corrigé, pavé numérique vérifié, encodage, anti-double tap.
