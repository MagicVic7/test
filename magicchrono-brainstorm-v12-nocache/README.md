# Magic Chrono - Brain Storm V12 NoCache

Objectif : revenir à une version claire qui ne reste pas bloquée sur une ancienne version PWA.

- Affiche `V12` sur l'écran PIN Brain Storm.
- Supprime les anciens service workers et caches au chargement.
- Lit le format exact : `TargetPosition`, `birthday`, `full_peek`, `text`, `timeUpdated`.
- `birthday` accepte `(29/06)` et `29/06`.
- Ouvre le site avec `?v=12` pour contourner l'ancien cache.
