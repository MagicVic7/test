# Magic Chrono - Brain Storm Manual V13

Autre approche fiable, sans API.

## Mode position

- Ouvre Brain Storm.
- Le PIN affiche `Format PPxx`.
- Tape `3900` pour forcer la position 39 au STOP.
- Les 2 derniers chiffres sont du camouflage.

## Mode anniversaire

- Sur l'écran PIN, appuie sur **Appel d’urgence**.
- Le PIN passe en mode `PPJJMM`.
- Exemple : `392906` signifie :
  - position 39 au STOP ;
  - premier Tour = 29 ;
  - deuxième Tour = 06.

Cette version désactive aussi le service worker pour éviter les caches PWA.
